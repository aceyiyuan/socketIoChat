Quick start

- create virtual environment

  Linux and OS x:
  
  ```
  $ python3 -m venv myvenv
  $ source myvenv/bin/activate
  ```

- run 

  ```
  $ pip install -r requirements.txt
  ```
- start server 

  ```
  $ python app.py

  ```
